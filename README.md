# Goal-Based Agent (Job Application Assistant)

This project contains two versions of a **goal-based conversational agent** built with LangChain.  
The agent’s goal is to collect a user’s **name, email, and skills** using tools and memory.

## 🚀 Features
- Built with **LangChain** and **OpenAI/Ollama/Google Gemini** integrations.
- Uses **ConversationBufferMemory** to track chat history.
- Two versions:
  - `goal_based_agent_v1.py` → baseline implementation
  - `goal_based_agent_v3.py` → extended/experimental version
